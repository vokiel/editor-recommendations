{% capture markup %}

The `<s>` element MUST be used to represent the **Strikethrough** feature as it is intended to mark parts of content that are no longer relevant<sup>[[1](#ref1)]</sup>.

Example:

```html
HTML 4 <s>is</s> was the best HTML ever.
```
